## Overview
BalanceBreak Dashboard is a **frontend-only web dashboard** designed to help balance productivity and breaks.  
It combines study-friendly activities, exercises, and fun breaks into one simple, interactive interface.

This is a **personal  project** focused on JavaScript, APIs, and dynamic UI behavior.

---

## Features
- Categorized activity cards
- Exercise data.
- Break activities (games, movies, outdoor ideas)
- Dynamic rendering and filtering
- Axios with async/await
